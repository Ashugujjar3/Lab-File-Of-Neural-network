This repository contains the practical implementations and documentation for the **Neural Networks Lab** course. It includes various algorithms and models studied as part of the curriculum, focusing on artificial intelligence and machine learning fundamentals.

The experiments in this lab were implemented using:

* **Core Libraries:** NumPy (for matrix operations), Matplotlib (for data visualization)
* **Frameworks:** Scikit-learn, TensorFlow/Keras (where applicable)
